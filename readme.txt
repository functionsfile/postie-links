An add-on for the [Postie](https://wordpress.org/plugins/postie/) plugin. Creates a post with the "Link" format if the email content only contains a URL.

== Description ==

The Postie Links Add-On extends the [Postie](https://wordpress.org/plugins/postie/) plugin. Create new posts with the 'link' post format by sending an email with a URL as the email body.

Do you want to report a bug or suggest a feature for Postie Links Add-On? Best to do so in the [Postie Links Add-On repository on GitHub](https://github.com/functionsfile/postie-links/).

== Installation ==

To use Postie Links Add-On, there are only two things you need to do: install and activate the plugin. Postie Links Add-On also works great as a [must-use plugin](https://wordpress.org/documentation/article/must-use-plugins/).

### Install Postie Links Add-On from within WordPress

1. Visit the Plugins page within your dashboard and select ‘Add New’
1. Search for 'Postie Links Add-On'
1. Install and activate Postie Links Add-On from your Plugins page

### Install Postie Links Add-On manually

1. Upload the 'post-links' folder to the /wp-content/plugins/ directory
1. Activate the Postie Links Add-On plugin through the 'Plugins' menu in WordPress
